## Utilizar en ambiente local

### Correr el siguiente comando en la terminal una vez se encuentre ubicado en la carpeta del proyecto

```
docker-compose up --build
```  

## Bajar el contenedor

### Correr el siguiente comando en la terminal una vez se encuentre ubicado en la carpeta del proyecto

```
docker-compose down
```
